** Using Branch

$ git branch testing

Point existing HEAD to branch testing

$ git checkout testing

              Head
               |
               V
[C0] - [C1] - [C2]

Following two are equivalent:

Ver1: 
$ git checkout -b iss53

Ver2:
$ git branch iss53
$ git checkout iss53
 
Merge the code together:
$ git checkout master
$ git merge hotfix



** Using Magit with Emacs

Download Magit, latest at this moment is 1.2.

Key commands
 - magit-status
 In Status Mode:
    "c" to commit
    "g/G" to refresh/refresh all
    "s" to stage the
 - magit-log
 - magit-diff
 - C-c C-c to confirm commit
 - C-c C-k to cancel commit
